# Launch-Android-Studio-Emulator-CMD
How to launch the Android Studio Emulator from CMD

Windows:
1. cd C:\Users\%USERNAME%\AppData\Local\Android\Sdk\emulator
2. emulator -list-avds
3. emulator -avd <avd_name> eg. (emulator -avd Pixel_5_API_30).

Mac:
1. emulator -avd Pixel_5_API_30
2. emulator -list-avds
3. emulator -avd <avd_name> eg. (emulator -avd Pixel_5_API_30).
